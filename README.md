# Execução do projeto

No diretório ../trustion-portal, execute:

1) Download de depedências

```
npm i (ou npm install)
```

2) Inicialização do servidor

```
npm run httpserver
```

O endereço http://localhost:4200/#!/login será carregado automaticamente no navegador.
